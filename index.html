hexo博客搭建和部署

准备工作

1.安装node.jsnode下载地址

2.安装git

3.安装hexo
$ npm install hexo-cli -g

开始搭建

1.新建一个项目文件夹。例：hexo-blog

2.在项目文件（hexo-blog）目录下运行git bash,输入以下命令：
$ hexo init

3.项目初始化完成后就可以运行项目了，输入以下命令：
$ hexo server
该命令可以简写为
$ hexo s
运行成功会给你一个本地地址，复制到浏览器地址栏打开即可。(http://localhost:4000)

4.新建一篇文章
$ hexo new “我的第一篇文章”
该命令可以简写为
$ hexo n “我的第一篇文章”
建议使用markdown语法编写内容，写完后使用命令生成静态文件：
$ hexo generate
该命令可以简写为
$ hexo g
然后在项目根目录下运行git输入hexo s运行项目进行预览

部署到GitHub

1.登录个人GitHub新建一个仓库，建议仓库名为：GitHub用户名.github.io

2.安装部署插件，在项目根目录下运行git输入：
